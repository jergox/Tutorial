    "serie":

            "episodios": 16,

Editar una serie por su id (en este caso id=1)

PUT:http://127.0.0.1:4000/serie/1 
{
    "estado": "eliminado",
    "serie":
        {
            "episodios": 16,
            "genero": "comedia",
            "id": 1,
            "img": "",
            "nombre": "What's Wrong with Secretary Kim?",
            "pais": "corea"
        }
}
